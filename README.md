# Python F1 Simulator

A text-based Formula 1 simulator written in Python, designed for people who love F1 and have too much free time! This project lets you simulate thrilling F1 races, complete with pitstops, crashes, and dynamic race events. Discover a world of over 150 fictional drivers and 30 constructors, making every race unique.

## Features

- **Text-Based Simulation:** Experience F1 racing in your terminal, with a final match grid.
- **Dynamic Events:** The simulator models pit stops, random crashes, retirements, and other race incidents.
- **Massive Driver & Constructor Pool:** Choose from 150+ fictional drivers and 30 constructors, each with unique stats.
- **Track Support:** Currently, the simulator features the Singapore 2025 Grand Prix. More tracks will be added in future updates.
- **Final Standings:** See full race results and standings after every simulation.
- **Future Plans:**
  - Multiple tracks and race calendar.
  - Full championship mode with points system, standings, and progression.


## Getting Started

### Prerequisites

- Python 3.7 or higher
-PrettyTable previously installed

### Running the Simulator

Clone this repository and run the simulator:

```bash
git clone https://github.com/yourusername/f1-simulator.git
cd f1-simulator
python main.py
```

Follow the on-screen instructions to set up a race and watch the action unfold!
